\chapter{FUTURE WORKS}

\section{Platform Enhancements}
\begin{itemize}
    \item \textbf{Real-time notifications:} Introduce websocket-based notifications for event approvals, team application updates, and recruitment post activity.
    \item \textbf{Advanced search and filters:} Add multi-criteria filtering for teams, events, and recruitment posts (game, rank, region, role).
    \item \textbf{Analytics dashboard:} Provide engagement and conversion metrics for administrators and recruiters.
\end{itemize}

\section{Security and Reliability}
\begin{itemize}
    \item \textbf{Database-portable migrations:} Refactor migrations to support both PostgreSQL and SQLite for smoother automated testing.
    \item \textbf{Automated regression suite:} Expand feature tests for event registration, team applications, and recruiter workflows.
    \item \textbf{Operational monitoring:} Add structured logs, health checks, and deployment smoke tests.
\end{itemize}

\section{User Experience Improvements}
\begin{itemize}
    \item \textbf{Profile enrichment:} Add media-rich player profiles with verified statistics and match history.
    \item \textbf{Team communication tools:} Provide in-app announcements and invitation workflows with audit trails.
    \item \textbf{Accessibility pass:} Improve keyboard navigation, contrast, and screen-reader support across dashboards.
\end{itemize}
